# Project Overview: Faktura do KSeF - Landing Page

## What is this?

This is a Landing Page for a SaaS application. The main application (already in production) allows users to upload multi-item PDF invoices (e.g., from insurance companies) and automatically converts and sends them to the Polish government tax system (KSeF). The difference is that the production app DOES NOT USE AI, so the conversion is very fast and data are safe - not send anywhere!

## Architecture

- **This Repository:** The marketing Landing Page (SEO optimized, broad audience targeting).
- **Tech Stack:** Next.js (App Router), Tailwind CSS, static export.
- **The Main App:** Lives on a separate subdomain (`app.faktura-do-ksef.pl`).
- **Goal:** The Landing Page must explain the value proposition, rank well in Google (SEO), and drive users to click the "Register/Login" buttons which will redirect them to the external app subdomain.

## Target Audience

Primary: Tow truck companies (Pomoc drogowa, Lawety) dealing with insurance invoices.
Secondary: Car repair shops, B2B companies with complex invoices.

## Design Guidelines

- Modern, clean SaaS look.
- Trustworthy and professional.
- Simple to read for older/non-tech-savvy users.
